# Gurugram Real Estate Analysis

## 📌 Project Overview
This project analyzes Gurugram real estate data using Python to identify pricing trends, locality insights, and property patterns.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset
Real estate dataset containing:
- Price
- Area
- Locality
- Property type
- Rate per sqft

## 📈 Key Insights
- Identified costliest localities
- Compared ready-to-move vs under-construction prices
- Found highest price-per-square-foot areas
- Analyzed property trends
